Gymshark Dashboard

Financial Statements and Visuals

This dashboard is the product of smart Excel data modelling and Power Query.
Why did i make this?
1. To improve my data modelling and BI skills.
2. To *hopefully* get a job at Gymshark (Pending as of May 2022)

How did i make this? - The Basic Setup
1. Extract Financial Statements and associated schedules from Gymshark Annual Reports (2016 to 2020). This data is available at Companies House
2. Build dynamic Excel model using schedules.
3. Build data model template for use in Power BI. Indexing is a key method for converting Excel financial statements into a format for PowerBI
4. Import Excel data models into PBI Power Query and Transform it.
5. Use simple matrix visual for the Financial Statements.
6. Upload final report to PowerBI Service and set up automated refresh schedule (every 24 hours)

There is a lot of DAX that makes this report dynamic and automated in addition to the "Basic Setup". I will release a webinar tutorial on this topic in the future.
